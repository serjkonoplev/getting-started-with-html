World Wide Web (www)

www - is a collection of HTML documents. This documents are made of HTML (HyperText Markup Language).
To see this documents i send HTTP Request to a Server via the internet. The main instrument i use to recieve and interpret HTML pages is a browser.



HTML "glues" all types of information (videos,music,pictures etc.).
It contains of :
-Text content which describes what you see
-Markup which describes what it looks like
-References to other documents such as video, music etc.
-Links to other pages

Inline vs Block Element

HTML elements may be inline or block. Block elements form an "invisible box" around the content inside of them.
